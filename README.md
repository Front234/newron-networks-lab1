## Лабораторная работа "Введение в нейронные сети"

## Структура репозитория

- `ДЗ_Lite.ipynb` – ноутбук для задания Lite: перебор гиперпараметров (количество нейронов, функция активации, размер батча) на наборе геометрических фигур (треугольник, круг, квадрат).
- `ДЗ_Pro версия 1.ipynb` – ноутбук для задания Pro: бинарная классификация изображений крестиков и ноликов. 
- `ДЗ_Ultra_Pro.ipynb` – ноутбук для задания Ultra Pro: распознавание рукописной цифры, нарисованной от руки. Модель обучается на MNIST, затем предсказывает цифру с загруженного пользователем фото.
- `pytorch lab 1.ipynb` – ноутбук для задания Lite: перебор гиперпараметров (количество нейронов, функция активации, размер батча) на наборе геометрических фигур (треугольник, круг, квадрат). Выполнено с использованием Pytorch

## Требования

Для запуска ноутбуков рекомендуется использовать **Google Colab**. Все необходимые библиотеки предустановлены в Colab.
